# Mini Projects

A collection of beginner-friendly mini projects built with `HTML`, `CSS`, and `JavaScript`.

Each project lives in its own folder and includes:

- `index.html`
- `style.css`
- `script.js`

These projects are meant for practice, revision, and quick experimentation while learning front-end basics.

## Project List

1. `01_counter-app` - Increase, decrease, and reset a counter.
2. `02_theme-switcher` - Switch between light, dark, and ocean themes.
3. `03_digital-clock` - Displays the current time live.
4. `04_color-picker` - Pick a color and view its hex value.
5. `05_quote-generator` - Shows a random quote on button click.
6. `06_todo-list` - Add, complete, and delete tasks.
7. `07_calculator` - Performs basic arithmetic operations.
8. `08_password-generator` - Generate a random password.
9. `09_weather-app` - A simple demo weather app with city input.
10. `10_image-slider` - Browse through a set of sample images.
11. `11_quiz-app` - Answer questions and see your score.
12. `12_tic-tac-toe` - A simple two-player tic-tac-toe game.

## How To Use

1. Open any project folder.
2. Launch `index.html` in your browser.
3. Edit the files to experiment and learn.

## Goal

The goal of this repository is to build small, easy projects consistently and improve core web development skills through practice.
